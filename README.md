## Tech Stack

- React.js (Vite)
- Node.js
- Express.js
- MongoDB Atlas
- Axios
- CSS

## Features

- Contact form with validation
- Save contacts to MongoDB
- Fetch contacts without page reload
- Success popup after submit
- Responsive centered layout

## API Endpoints

GET /api/contacts  
POST /api/contacts

## How to Run Locally

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev
